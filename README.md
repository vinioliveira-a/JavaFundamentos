# Desafios de Fundamentos em [Linguagem de Programação]

## Descrição do Projeto

Este repositório contém uma série de exercícios práticos focados nos fundamentos da programação, projetados para solidificar conceitos básicos como entrada/saída de dados, operações aritméticas, condicionais e manipulação de variáveis. Cada desafio aborda um problema específico, incentivando a aplicação de lógica de programação para resolvê-los.

## Funcionalidades

Os programas desenvolvidos neste projeto incluem as seguintes funcionalidades:

- **Conversão de Temperatura (Fahrenheit para Celsius)**: Um programa que solicita uma temperatura em Fahrenheit e a converte para Celsius, exibindo o resultado.

- **Conversão de Temperatura (Celsius para Fahrenheit)**: Um programa que solicita uma temperatura em Celsius e a converte para Fahrenheit, exibindo o resultado.

- **Cálculo de IMC**: Um programa que lê o peso (em kg) e a altura (em metros) do usuário e calcula o Índice de Massa Corporal (IMC), apresentando o valor no console.

- **Cálculo de Quadrado e Cubo**: Um programa que lê um valor numérico e exibe o resultado desse valor elevado ao quadrado e ao cubo.

- **Cálculo da Área do Triângulo**: Um programa que solicita a base e a altura de um triângulo e calcula sua área, utilizando a fórmula `(base * altura) / 2`.

- **Resolução de Equações do Segundo Grau (Bhaskara)**: Um programa que resolve equações do segundo grau no formato `ax² + bx + c = 0` utilizando a fórmula de Bhaskara. O programa deve calcular o discriminante (delta) e as raízes da equação. Um exemplo de uso é com `a = 1`, `b = 12` e `c = -13`.

## Tecnologias Utilizadas

- **[Java]**: Linguagem principal utilizada para o desenvolvimento dos exercícios.

## Estrutura do Código

Cada exercício é implementado em um arquivo separado, seguindo uma estrutura clara e concisa. Os programas geralmente envolvem:

- Leitura de dados de entrada do usuário.

- Processamento lógico e cálculos.

- Exibição dos resultados no console.

## Como Executar Localmente

Para executar estes programas, siga os passos abaixo:

1. Certifique-se de ter o ambiente de desenvolvimento para **[Java]** configurado em sua máquina.

1. Clone o repositório para a sua máquina local:

   ```bash
   git clone [https://github.com/vinioliveira-a/JavaFundamentos.git]
   ```

1. Acesse a pasta do projeto:

   ```bash
   cd cesafioBasicos
   ```

1. Navegue até o diretório do exercício desejado e execute o programa conforme a linguagem utilizada (ex: `java NomeDoArquivo.java` ou `python NomeDoArquivo.py`).# JavaFundamentos
